最近「龙虾」不是很火嘛，腾讯也搞了一个，叫 QClaw，就是腾讯版的龙虾。跟原版的差不多，该有的都有。

然后我想，既然要写 HEXO 博客，干脆整个自动化流程出来得了。这个流程大概是这样的：

首先，告诉 AI 「帮我创建一个 HEXO 博客技能」，它就把技能做出来。把 GitHub 仓库地址甩给它，让它把代码拉下来，做成可用的 Skill。

这个 Skill 有什么用呢？以后写博客的时候，流程就变成：

1. 跟 AI 说「帮我写一篇博客」，说明主题
2. AI 帮你把文章写好，自动加上 Frontmatter
3. Typora 打开给你看，哪里要改直接在 Typora 里改
4. 改完确认，AI 自动帮你推送到 GitHub 仓库
5. GitHub 那边有 Actions 自动触发构建和部署

整个链路就这样打通了，写作、预览、推送、部署都不用手动操心。

所以最后的效果就是，手机上直接跟 AI 说「帮我写一篇博客」，剩下的全部自动完成。
